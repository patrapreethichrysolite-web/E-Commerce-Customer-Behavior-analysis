E-Commerce Customer Behavior Analysis

 Overview

This project analyzes customer behavior in an e-commerce platform using Excel. The goal is to identify spending patterns, customer satisfaction levels, membership performance, discount impact, and city-wise trends.

Dataset

The dataset contains customer demographics, membership details, purchase information, discounts, and satisfaction levels.

Tools Used

* Microsoft Excel
* Pivot Tables
* Charts & Dashboard

 Analysis Performed

* Membership Type vs Average Total Spend
* Gender vs Average Total Spend
* Satisfaction Level Distribution
* Discount Applied vs Average Total Spend
* City vs Average Total Spend

Key Insights

* Identified the highest-spending membership category.
* Analyzed customer spending based on gender.
* Evaluated customer satisfaction levels.
* Measured the impact of discounts on spending.
* Compared spending patterns across cities.
